1. Вкладка «Расчет штампа» (расчёт стоимости ножевого штампа)
Назначение:
Автоматический расчёт стоимости вырубного штампа (ножа) для тиснения/вырубки упаковки на основе суммарной длины режущих кромок и выбранных диаметров пробивных отверстий.

Основные функции:

Ввод длины ножей – вручную (в миллиметрах) или автоматически через интеграцию с Adobe Illustrator.
Если в Illustrator выделены объекты, программа вычисляет их суммарную длину (с учётом вложенных групп) и подставляет значение в поле.

Выбор пробивки (отверстий) – диаметры 4, 5, 6–8, 9–10 мм. Для каждого можно указать нужное количество штук; стоимость за единицу задана (от 360 до 450 руб.).

Базовая стоимость штампа – 4300 руб + 1700 руб за каждый миллиметр длины.

Итоговая цена – суммируется базовый расчёт и стоимость выбранной пробивки.

Дополнительно:

Отображение статуса подключения к Illustrator (✅ Запущен / ❌ Не запущен).

Контекстное меню и горячая клавиша Ctrl+C для копирования результата.

Всплывающее уведомление при вставке значения из Illustrator.

2. Вкладка «Расчет вкладышей» (оптимизация размещения на листе)
Назначение:
Подбор наилучшего расположения вкладышей (этикеток, карточек) на печатном листе с учётом формата бумаги, типа печати, цветности, отступов и возможности переворота.

Основные функции:

Выбор формата бумаги:

Стандартные размеры: 620×940, 640×900, 700×1000, 720×1040 мм.

Произвольный размер (с проверкой ограничений: высота ≥360 мм, ширина ≤1038 мм).

Задание размера вкладыша – ширина и высота в мм.

Тип печати и оборота:

Односторонняя.

Двусторонняя «Чужой оборот» (переворот всего листа).

Двусторонняя «Свой оборот» (лицом к лицу – требуется чётное количество по ширине).

Цветность:

Цветная – добавляется припуск 4 мм.

Чёрно-белая – без припусков.

Опции оптимизации:

Автоматический переворот вкладыша (если так помещается больше).

Автоматический расчёт для всех стандартных форматов и сравнение.

Режим сжатия – уменьшение отступов (25 → 22 мм) и/или припусков для более плотной укладки.

Результаты:

Детальный отчёт в текстовом поле: расположение (ряды × колонки), итоговое количество на листе (с учётом особенностей оборота), эффективность использования площади.

Краткая сводка: лучший формат, оптимальное расположение, общий размер блока вкладышей.

Таблица сравнения всех форматов с сортировкой по количеству вкладышей.

3. Вкладка «Расчет % выгоды» (сравнение двух листов)
Назначение:
Оценка, насколько один вариант размеров листа выгоднее другого по площади (в процентах). Полезно при выборе поставщика картона или оптимизации раскроя.

Основные функции:

Ввод четырёх значений:

Высота и ширина листа №1.

Высота и ширина листа №2.

Расчёт:
Вычисляются площади обоих листов. Определяется, какой вариант меньше, и считается процент разницы относительно бóльшего листа.

Вывод:
Текст вида: «На: 12.34 % выгоднее второй вариант 640x900».

4. Вкладка «Расчет тиража» (подбор пачек на лист)
Назначение:
Расчёт минимального количества пачек продукции, которое можно разместить на одном печатном листе, чтобы покрыть заданный тираж с округлением до чётного числа (требование некоторых печатных машин/фальцовок).

Основные функции:

Ввод:

Общий тираж (количество пачек).

Количество знаков (пачек), которое помещается на листе по технологии (обычно даёт технолог).

Алгоритм:

Результат округляется до ближайшего чётного вверх.

Сравнивается с введённым количеством знаков: если мало – выводится подсказка, сколько добавить.

Рассчитывается число листов и итоговое количество пачек.

Вывод:
Результат в многострочном виде:

Минимальное количество пачек на лист.

Листов всего.

Итоговое количество пачек.

Общая интеграция с Adobe Illustrator
Только вкладка «Расчет штампа» постоянно отслеживает выделение в Illustrator (если он запущен).

При изменении выделения длина автоматически подставляется в поле «Введите длину ножей».

В правом верхнем углу вкладки отображается статус подключения к Illustrator.

Остальные вкладки работают независимо от Illustrator.
